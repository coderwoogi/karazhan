# Karazhan Addon Studio

드래그 앤 드롭으로 WoW 애드온 UI 레이아웃을 설계하고, JSON과 Lua로
내보내는 1차 편집기입니다.

현재 포함 기능:
- 툴박스에서 위젯 드래그 추가
- 속성 편집
- JSON 저장 / 열기
- Lua 내보내기 / 복사

1차 위젯:
- Label
- Panel
- Button
- ProgressBar

향후 계획:
- 앵커 편집
- 폰트 / 정렬 / 텍스처 에셋 선택
- 서버 바인딩 키 자동 생성
- `mod-instance-bonus-mission` 실사용 내보내기 템플릿
