암상인 모듈 정리

암상인 모듈의 기능 개요와 사용 중인 데이터베이스 테이블, 관리자 명령어를 한 문서에서 확인할 수 있도록 정리한 참조 문서입니다.

이름 암상인 시스템
설명 암상인을 월드 내 특정 후보 위치 중 한 곳에 생성하고, 등록된 아이템 풀에서 일부를 골라 판매하며, 구매 로그와 현재 상태를 저장하는 모듈입니다. 현재 로직은 구매가 발생할 때만 다른 랜덤 위치로 이동합니다.
모듈명 mod-blackmarket
파일 위치 E:\server\azerothcore-wotlk\modules\mod-blackmarket

사용 테이블

암상인 모듈이 실제로 사용하는 월드/캐릭터 데이터베이스 테이블과 역할입니다.

데이터베이스 테이블명 유형 설명 주요 내용 파일 위치
acore_world blackmarket_spawn_points Table 암상인이 생성될 수 있는 후보 위치 목록 컬럼: id, map, position_x, position_y, position_z, orientation, zone_id, comment E:\server\azerothcore-wotlk\modules\mod-blackmarket\data\sql\db-world\blackmarket_spawn_points.sql
acore_world blackmarket_item_pool Table 암상인이 판매 후보로 사용할 전체 아이템 풀 컬럼: id, item_entry, price_gold, price_item, price_item_count, rarity, weight, max_per_spawn, comment E:\server\azerothcore-wotlk\modules\mod-blackmarket\data\sql\db-world\blackmarket_item_pool.sql
acore_world blackmarket_vendor_items Table 현재 세션에서 암상인이 실제로 판매 중인 아이템 목록 컬럼: id, item_entry, price_gold, price_item, price_item_count, remaining_count, comment E:\server\azerothcore-wotlk\modules\mod-blackmarket\data\sql\db-world\blackmarket_vendor_items.sql
acore_world blackmarket_state Table 암상인의 현재 활성 상태와 현재 위치 정보를 저장하는 단일 상태 테이블 컬럼: id, is_active, creature_guid, spawn_point_id, spawn_time, despawn_time E:\server\azerothcore-wotlk\modules\mod-blackmarket\data\sql\db-world\blackmarket_state.sql
acore_characters blackmarket_purchase_log Table 누가 어떤 아이템을 언제 구매했는지 기록하는 로그 테이블 컬럼: id, account_id, character_guid, item_entry, purchase_time E:\server\azerothcore-wotlk\modules\mod-blackmarket\data\sql\db-characters\blackmarket_purchase_log.sql

관리자 명령어

암상인 모듈에 연결된 GM/관리자용 명령어 목록입니다.

실행 환경 명령어 유형 설명 권한 및 동작 파일 위치
worldserver .blackmarket tele add <comment> Command 현재 GM 위치를 암상인 스폰 포인트로 추가 권한: SEC_GAMEMASTER, blackmarket_spawn_points 테이블에 INSERT 수행 E:\server\azerothcore-wotlk\modules\mod-blackmarket\src\BlackMarketCommands.cpp
worldserver .blackmarket enable Command 암상인 시스템을 활성화 권한: SEC_ADMINISTRATOR, 비활성 상태면 Enable 호출 E:\server\azerothcore-wotlk\modules\mod-blackmarket\src\BlackMarketCommands.cpp
worldserver .blackmarket disable Command 암상인 시스템을 비활성화 권한: SEC_ADMINISTRATOR, 활성 상태면 Disable 호출 E:\server\azerothcore-wotlk\modules\mod-blackmarket\src\BlackMarketCommands.cpp
worldserver .blackmarket toggle Command 암상인 시스템 활성/비활성 상태를 전환 권한: SEC_ADMINISTRATOR E:\server\azerothcore-wotlk\modules\mod-blackmarket\src\BlackMarketCommands.cpp
worldserver .blackmarket status Command 암상인 시스템 상태를 출력 권한: SEC_GAMEMASTER, 시스템 활성 여부와 NPC 스폰 여부, 세션 ID 출력 E:\server\azerothcore-wotlk\modules\mod-blackmarket\src\BlackMarketCommands.cpp
worldserver .blackmarket go Command 현재 스폰된 암상인 위치로 이동 권한: SEC_GAMEMASTER, 현재 merchant GUID를 찾아 TeleportTo 수행 E:\server\azerothcore-wotlk\modules\mod-blackmarket\src\BlackMarketCommands.cpp
