# mod-solo-arena

달라란 투기장을 사용하는 1인 아레나 챌린지 모듈입니다.

v1 범위:
- 전용 NPC와 대화로 시작
- 1~3단계 선택
- 달라란 투기장으로 개별 인스턴스 이동
- 플레이어를 닮은 그림자 bot 소환
- 단계별 체력/피해/공격속도/스킬 빈도/이동속도 강화
- 승리/패배 시 자동 종료 및 복귀

DB:
- world: `solo_arena_stage`, 커스텀 NPC/그림자 템플릿
- characters: `solo_arena_progress`, `solo_arena_run_log`

주의:
- v1의 bot은 "같은 외형 + 직업 기반 전투 패키지"입니다.
- 플레이어 특성과 100% 동일한 회전/판단까지는 아직 구현하지 않습니다.
