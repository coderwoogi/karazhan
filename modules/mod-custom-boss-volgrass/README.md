# Volgrass Boss Module

침공자 볼그라스 보스 전투 모듈입니다.

## 📋 개요

볼그라스는 3단계 페이즈 시스템을 갖춘 침공형 보스로, 독특한 기믹과 **풍부한 랜덤 멘트 시스템**을 통해 매번 다른 전투 연출을 제공합니다.

## 🎭 멀티 멘트 시스템

### ✨ 특징
- **creature_text 테이블 활용**: AzerothCore 표준 방식
- **모든 이벤트에 3~6개의 멘트 풀 적용**
- **Talk(GroupID)로 자동 랜덤 선택** - 코드에서는 GroupID만 지정하면 DB에서 자동으로 랜덤 선택
- **Phase별 차별화된 멘트 강도** (Phase 3에서 더욱 위협적)
- 기획 단계에서 정의된 핵심 멘트("모두 이리로 와라", "도망쳐라, 가능하다면", "섬과 함께 사라져라" 등) 모두 포함

### 📊 creature_text GroupID 매핑

| GroupID | 이벤트 | 멘트 개수 | 사용 시점 |
|---------|--------|-----------|-----------|
| 0 | 전투 시작 | 5개 | 공격 개시 시 |
| 1 | Phase 2 진입 | 5개 | 70% HP 이하 |
| 2 | Phase 3 진입 | 6개 | 40% HP 이하 (폭주) |
| 3 | 죽음의 샘 | 6개 | 끌어당김 경고 ⭐ |
| 4 | 대폭발 (일반) | 6개 | Phase 1~2 폭발 ⭐ |
| 5 | 대폭발 (Phase 3) | 6개 | Phase 3 전용 위협적 폭발 ⭐ |
| 6 | 충전의 도약 | 5개 | 랜덤 타겟 도약 |
| 7 | 초강력 열풍 | 5개 | 광역 바람 공격 |
| 8 | 하수인 소환 | 5개 | Phase 2/3 소환 |
| 9 | 플레이어 처치 | 6개 | 플레이어 사망 시 |
| 10 | 보스 사망 | 5개 | 보스 패배 시 |

**총 68개의 고유 멘트**가 전투 중 랜덤하게 출력됩니다!

### 📢 주요 멘트 예시

#### 핵심 경고 멘트 ⭐
- **죽음의 샘**: "모두 이리로 와라!", "도망칠 곳은 없다!" (회피 준비 신호)
- **대폭발**: "도망쳐라, 가능하다면!", "재가 되어라!" (즉시 회피)
- **Phase 3 대폭발**: "섬과 함께 사라져라!", "우주의 먼지가 되어라!" (최종 경고)

#### Phase별 분위기
- **Phase 1**: 여유 있는 위협 ("폭풍 속에서 무릎 꿇어라!")
- **Phase 2**: 본격적인 공격 ("이제 진짜 힘을 보여주마!")
- **Phase 3**: 폭주 상태 ("너희 모두... 섬과 함께 사라져라!")

## ⚔️ 페이즈 시스템

### Phase 1 (100-70% HP)
- 초강력 열풍 (15~20초 주기)
- 충전의 도약 (18~25초 주기)
- 기본 전투 패턴 학습 단계

### Phase 2 (70-40% HP)
- **죽음의 샘 + 대폭발 콤보 시작** (핵심 기믹)
- 하수인 소환 활성화
- 콤보 주기: 25~30초

### Phase 3 (40-0% HP)
- **폭주 상태 진입**
- 죽음의 샘→대폭발 콤보 주기 단축 (**15~20초**)
- 더 많은 하수인 소환
- **Phase 3 전용 위협적 멘트** 사용

## 🎯 핵심 기믹: 죽음의 샘 → 대폭발 연계

```
1. 죽음의 샘 시전
   └─> 보스: "모두 이리로 와라!" (GroupID 3 중 랜덤)
   └─> 50m 범위 내 모든 플레이어 끌어당김
   
2. 1.5~2초 대기
   └─> 회피 타이밍
   
3. 대폭발 시전
   └─> Phase 1~2: "도망쳐라, 가능하다면!" (GroupID 4)
   └─> Phase 3: "섬과 함께 사라져라!" (GroupID 5)
   └─> 광역 피해 발생
```

**전략**: 끌어당김 멘트를 듣는 즉시 산개 준비, 폭발 멘트가 나오면 최대한 멀리 회피!

## 🛠️ 설정 옵션

`mod_custom_boss_volgrass.conf` 파일:

```conf
# 하수인 몬스터 Entry ID (0 = 비활성화)
Volgrass.Add.EntryID = 0

# Phase 2 소환 개수
Volgrass.Phase2.AddCount = 2

# Phase 3 소환 개수
Volgrass.Phase3.AddCount = 4

# 소환 주기 (밀리초)
Volgrass.Summon.Interval = 30000
```

## 💡 공략 팁

### 생존 전략
1. **죽음의 샘** 멘트 ("모두 이리로 와라!" 등) 즉시 산개 시작
2. **대폭발** 멘트 ("도망쳐라, 가능하다면!" 등) 최대 거리 확보
3. Phase 3에서 "섬과 함께 사라져라!" 멘트는 **최우선 회피**

### Phase별 전략
- **Phase 1**: 패턴 익히기, 딜 집중
- **Phase 2**: 끌어당김→폭발 콤보 대응 연습, 하수인 우선 처리
- **Phase 3**: 모든 스킬 주기 단축됨, **생존 최우선**

### 역할별 가이드
- **탱커**: 보스 위치 고정, 끌어당김 후 신속 복귀
- **딜러**: 대폭발 회피 최우선, 하수인 신속 제거
- **힐러**: Phase 3에서 힐 부담 급증, 마나 관리 중요

## 🔧 설치 가이드

상세 설치 방법은 `INSTALL.md` 참조

### 간단 설치

1. **모듈 복사**
   ```bash
   # modules 디렉토리에 복사
   ```

2. **CMake 재생성**
   ```bash
   cd build
   cmake ../ -DCMAKE_INSTALL_PREFIX=path/to/server
   ```

3. **빌드**
   ```bash
   cmake --build . --config RelWithDebInfo --target install
   ```

4. **SQL 실행**
   ```sql
   -- boss_volgrass.sql 실행
   -- creature_template, creature, creature_text 생성됨
   ```

5. **설정 복사**
   ```bash
   cp conf/mod_custom_boss_volgrass.conf.dist etc/mod_custom_boss_volgrass.conf
   ```

## 🔍 기술 사양

### AI 구조
- **EventMap 기반**: 모든 스킬과 페이즈 전환 관리
- **Phase 진입 제어**: bool 플래그로 1회만 진입 보장
- **creature_text 활용**: DB 기반 멀티 멘트 시스템

### 멘트 시스템 구현
- **C++ 코드**: `Talk(GroupID)` 호출만으로 랜덤 선택
- **DB (creature_text)**: 각 GroupID별 여러 개의 멘트 등록
- **자동 랜덤**: AzerothCore가 자동으로 랜덤 선택하여 출력

### 서버 공지
- `ChatHandler::SendWorldText()` 사용
- Phase 전환 시 전체 서버 공지
