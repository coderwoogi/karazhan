# Volgrass Island Invasion Master Plan

## 1. 문서 목적
- 기존 `mod-custom-boss-volgrass` 전투 기획(3페이즈 보스전)을 유지
- 신규 요구사항(2시간마다 정시 출몰, 20분 제한, 섬 침공 컨텐츠)을 통합
- 구현/밸런싱/운영에 바로 사용할 수 있는 기준 문서 제공

---

## 2. 현재 모듈 기준 요약 (기존 기획)

### 2.1 핵심 구조
- 보스 스크립트: `boss_volgrass`
- 전투 방식: 3페이즈 (`100-70`, `70-40`, `40-0`)
- 주요 구성:
  - `EventMap` 기반 스킬 루프
  - `Talk(GroupID)` 기반 랜덤 대사
  - `SendWorldText` 기반 전역 공지
  - 설정값 기반 졸몹 소환 (`Volgrass.Add.EntryID` 등)

### 2.2 기존 강점
- 페이즈 전환과 대사 연출이 이미 잘 분리되어 있음
- Death Grip -> Explosion 콤보로 보스 아이덴티티가 명확함
- 설정 파일(`.conf`)로 일부 난이도 튜닝 가능

### 2.3 기존 한계
- 침공이 실제 월드 이벤트라기보다 "풀링 후 보스전"에 가까움
- 출몰 시간 고정/실패 조건/이벤트 상태 관리가 없음
- 20분 제한, 점령/철수 연출, 웨이브 운영 시스템이 없음

---

## 3. 통합 목표 (신규 요구 반영)
- 특정 섬 침공 테마 강화
- 24시간 기준 2시간마다 정각 출몰
- 전투 시작 후 20분 내 미처치 시 보스 철수(실패)
- 3.3.5 기준 스킬을 활용한 개성 있는 패턴 구성
- 전투 중 다회 웨이브(졸몹 소환)로 "침공전" 체감 강화

---

## 4. 이벤트 전체 플로우

### 4.1 일일 사이클
1. `2시간 단위 정각 T`에 침공 이벤트 시작 (예: 00:00, 02:00, 04:00 ...)
2. 섬 전체 경보 메시지 + 전조 연출
3. 보스 출현 및 전투 개시
4. 20분 타이머 종료 전 처치 시 성공
5. 실패 시 보스 철수 + 웨이브 제거 + 이벤트 종료

### 4.2 상태 머신 (권장)
- `IDLE -> PREPARE -> BOSS_ACTIVE -> SUCCESS | FAIL -> COOLDOWN`

### 4.3 성공/실패 조건
- 성공: 20분 이내 Volgrass 처치
- 실패: 20분 초과 시 자동 철수(Despawn)

---

## 5. 보스 전투 설계 (기존 + 신규 통합)

### 5.1 페이즈 설계
1. Phase 1 (100-70%)
- 기존 기본기 유지: `Super Gale`, `Charged Leap`
- 신규: 산개 유도/위치 교정 패턴 1개 추가

2. Phase 2 (70-40%)
- 기존 콤보 강화: `Death Grip -> Big Explosion`
- 침공 웨이브 A/B 가동
- 광역 압박형 스킬 주기 추가

3. Phase 3 (40-0%)
- 기존 광폭 컨셉 유지(쿨다운 단축)
- 웨이브 C 가동
- 전장 압박(지속 광역 위협) 강화

### 5.2 20분 타임라인 (권장)
1. 00:00-02:00: 진입/탐색 구간
2. 02:00-08:00: Phase 2 + 웨이브 A
3. 08:00-15:00: Phase 3 + 웨이브 B
4. 15:00-20:00: 최종 압박 + 웨이브 C
